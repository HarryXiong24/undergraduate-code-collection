# 手机预览电脑页面

1.保证手机和电脑再同一个网络下（同一个wifi），

2.在你启动项目时候可以看到第二个地址，把地址生成一个二维码

![1586057884760](assets/1586057884760.png)

![1586057910584](assets/1586057910584.png)

3.手机扫描上面的二维码就可以看到你的开发页面

**4.如果手机访问不了需要关闭电脑的防火墙。**
